# Halla Taxi

Ride-hailing platform for Mombasa, Kenya — featuring rider app, driver app, and admin dashboard.

**Built with:** React + TypeScript + Tailwind CSS + Vite

## Quick Start

```bash
npm i
npm run dev
```

## Screens

- **Rider App** — Book rides, track drivers, M-Pesa payments
- **Driver App** — Accept requests, navigate, view earnings
- **Admin Dashboard** — Manage fleet, view analytics, pricing controls
